# XPLR Shop — Design Direction

## Aesthetic
Minimalist Premium. Dark, deep, luxurious. No clutter.

## Colors
- Background: deep navy-black `#060B18` → `#0A0F1C`
- Surface: `rgba(255,255,255,0.03)` to `rgba(255,255,255,0.06)`
- Text: white `#F8FAFC`, muted `#94A3B8`
- Accent price: gradient from cyan `#38BDF8` to violet `#A78BFA`
- Borders: `rgba(255,255,255,0.05)`
- Warning: amber `#F59E0B`
- Success: emerald `#10B981`

## Typography
- Display: "Plus Jakarta Sans" (weight 700–800)
- Body: "Plus Jakarta Sans" (weight 400–500)
- Monospace for prices: tabular-nums

## Spacing
- Generous padding, breathing room
- Mobile-first, responsive grid

## Motion
- Page transitions: fade + slide (motion library)
- Cards: subtle scale on hover
- Lists: staggered reveal on mount

## Components
- Hub cards: full-bleed images, gradient overlay, large text
- Country list: clean rows, flag + name + plan count, dividers at 5% white
- Product list: brand logo + name + description + price + buy button
- Modals: centered, backdrop blur, rounded corners
